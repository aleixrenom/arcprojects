# Project Instructions

A compact implementation guide for Copilot and this repo.

## Overview

Portfolio website for Aleix Renom Cisa with two content streams: apps and projects. The current UI is built around a two-page card grid and a full-screen detail view.

See `Design/DESIGN.md` for the full design spec.

## Tech Stack

- React + TypeScript + Vite
- Zustand for global state
- CSS custom properties for theming
- TanStack Router planned for shareable URLs
- Motion planned for animation

## Architecture

- `src/App.tsx` — root, renders either page grid or `DetailPage`
- `src/store/ui.ts` — `theme`, `activePage`, `selectedCard`, `openCard()`, `closeCard()`
- `src/pages/AppsPage.tsx` / `ProjectsPage.tsx` — card grids
- `src/pages/DetailPage.tsx` — blank full-screen detail page
- `src/index.css` — global theme and interaction styles
